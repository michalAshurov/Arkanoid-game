## About the Project
This project was built to strengthen my understanding of core software engineering concepts, including clean object-oriented design, event-driven programming, and real-time collision handling.

The game includes a paddle, ball, and breakable blocks, with interactive gameplay and visual customization.

## Key Features
- Object-Oriented Design (OOP)
- Collision detection and response
- Event listeners for keyboard input
- Modular and extensible game architecture
- Custom game loop and rendering logic

## Technologies

- Custom graphics and event handling (Java-based)
